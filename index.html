Полезные ссылки

Поскольку придумывать всё из головы — дело трудное и неблагодарное, у меня как и у любого уважающего себя дизайнера под рукой есть полезные ссылки и инструменты, к которым я прибегаю, чтобы ускорить и улучшить свою работу. Собственно, ниже следует список наиболее интересных ресусов в помощь дизайнеру. Пользуйтесь на здоровье и не сочтите за рекламу.

Фото знаменитостей, скрины, стиллы
  1. https://www.theplace.ru/photos/ или https://www.theplace2.ru/photos/
    Фото знаменитостей, в основном в хорошем качестве
  2. http://hq-pictures.com
    Ещё фото знаменитостей, групп, моделей, стиллы сериалов
  3. http://cultfilm.ru/
    Ещё немного разных фото с фотосессий. Возможно, это же есть и на двух ресурсах выше)
  4. http://www.farfarawaysite.com/
    Скрины, стиллы, промо и галереи актрис. Моё любимое) жаль, галерей мужчин-актёров нет(
  5. https://screencapped.net/
    Скрины из сериалов, фильмов. Требуется регистрация.

Клипарт, текстуры, фоны и т.п.
  1. https://pixabay.com/
    Без комментариев.
  2. https://pxhere.com/
    Чот среднее между Flickr и Pixabay XD
  3. https://www.cleanpng.com/ (раньше https://www.kisspng.com/)
    PNGшечки. Много PNGшечек, в т.ч. премиального качества.
  4. http://www.lenagold.ru/
    Клипарт, фоны и т.п. По отдельности и подборками.
  5. https://freestocktextures.com/
    Текстуры.
  6. https://www.freepik.com/
    Куча бесплатных векторых изображений.
  7. https://openclipart.org/
    Ещё клипарт.
  8. https://officialpsds.com/
    Ещё клипарт, в т.ч. люди.
  9. http://pngimg.com/
    И ещё png. png много не бывает XD
  10. http://www.stickpng.com/
    Клипарт по категориям.
  11. https://creativemarket.com/free-goods
    Разный клипарт, мокап, шрифты, градиенты и прочее в помощь творцам. Ресурсы платные, но если зарегаться, в этой категории (по ссылке) можно каждую неделю скачивать по 6 бесплатных ресурсов.
  12. https://www.creativefabrica.com/freebies/
    Похоже по смыслу на предыдущую ссылку — бесплатные товары на платном ресурсе. Есть те, которые не ограничены по времени. Требуется регистрация.
  13. https://unsplash.com/
    Фото, текстуры.

Спрайты, иконки
  1. https://www.spriters-resource.com/
  2. https://opengameart.org/
  3. http://spritedatabase.net/
    Спрайты моделей из игр (для создания кадров анимации и не только).
  4. https://icons8.com/
    Иконки разных наборов и стилей, можно выбрать свой размер.
  5. http://www.iconarchive.com/
    Архив иконок, в т.ч. по наборам.
  6. https://www.freeiconspng.com/
    Иконки png без регистрации и смс, впрочем, как и почти все остальные сайты в этом списке XD

Анимация
  1. https://giphy.com/
  2. https://gifer.com/en
    В целом 2 похожих ресурса, где можно найти гифки на любой вкус. Есть много красивой бесшовной анимации.

Инструменты, утилиты, уроки ФШ, полезные ресурсы для графистов
  1. https://photoshop-master.ru/
    Уроки ФШ, инструменты и т.п., в общем, дофига полезного по теме.
  2. https://pixelbox.ru/
    Тоже всякие уроки и инструменты ФШ.
  3. https://color.romanuke.com/
    Большое количество красивых палитр, возможность поиска палитр по 1 или 2м цветам.
  4. https://www.remove.bg/
    Сервис для удаления фона с фотографии (есть ограничения, но в целом годно).
  5. FastStone Image Viewer
    Программа для просмотра картинок, в т.ч. в формате gif (с проигрыванием анимации) и psd. Есть возможность делать мелкие манипуляции, вроде уменьшения размера, обрезки, скриншотов, добавления текста и обводки (если надо что-то выделить, допустим, для туториала).
  6. Free Video to JPG Converter
    Программа для нарезки скринкапсов (иными словами, для захвата скриншотов из фильма через определённый промежуток времени/количество кадров). Полностью бесплатная, никаких водных знаков и рекламы, интуитивно понятный интерфейс.
  7. Free Video to GIF Converter
    Простая программа для нарезки качественных мувов из любых видео.
    Урок по программе.

Инструменты и полезные утилиты для веба
  1. https://pages.github.com/
    Тут можно создать собственный сайт или отдельные html-страницы. Надеюсь, ресурс не подложит свинью, потому что я заипался уже искать хороший хостинг. Собстно, эта презентация лежит на моём гитхабе)
  2. http://angrytools.com/
    Неплохой инструментарий, где можно поэкспериментировать с css, например, с анимацией, градиентами, фильтрами.
  3. Sublime Text 3
    Хорошая прога для работы с html, css, js. Условно-бесплатная (пользоваться можно бесплатно, но иногда появляется раздражалка с просьбой приобрести программу). Позволяет просматривать создаваемый код в браузере. Можно подключить плагины для удобства. P.S. эта презентация писалась в данной проге :з
  4. http://htmlbook.ru/
    Справочник по html и css.
